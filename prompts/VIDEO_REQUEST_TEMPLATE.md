## 使い方

1. このテンプレートを記入
2. 必要な素材を `public/` 以下に配置
3. Claude Codeで以下のコマンドを実行:

```
YouTube風に編集して

[上記で記入した内容をペースト]
```

4. 自動で動画が生成され、`out/output.mp4` に保存されます
